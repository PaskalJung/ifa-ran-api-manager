Getting Started
Add collection http://SERVER_IP:SERVER_PORT/api/db/collection/add

GET

  • List all plants http://SERVER_IP:SERVER_PORT/api/db/list
  • Show plant by _id http://SERVER_IP:SERVER_PORT/api/db/index/:id id example: 5f26cf7c4290a047f50de001

POST

  • add plant http://SERVER_IP:SERVER_PORT/api/db/add send body example

PUT

  • update plant by _id http://SERVER_IP:SERVER_PORT/api/db/update/:id id example: 5f26cf7c4290a047f50de004 - send body example

DELETE

  • delete plant by _id http://SERVER_IP:SERVER_PORT/api/db/delete/:id id example: 5f26cf7c4290a047f50de004
